* uname : what you are running
* uname -r : get kernel version
* free -h
check free memory

* uname -a
kernel version, OS, hostname, CPU type, etc.

* lshw
 lists your hardware

* lspci
lists PCI hardware, which also included on-board devices

* du
(or ncdu)
disk space used by directories and files




* inxi -FGx
get system info
