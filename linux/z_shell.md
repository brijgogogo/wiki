= zsh =

* pacman -S zsh
* pacman -S zsh-completions

* /usr/bin/zsh
* chsh -s /bin/zsh
* ~/.zshrc : configuration setting
* ~/.zprofile : a script that will run upon login

* bindkey -v
to enable vi mode in .zshrc

== prompt ==
* prompt -l
* prompt <prompt_name>
* prompt -p
