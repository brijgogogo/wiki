= todo.txt cli =


* ~/.todo/config
config location

assuming t as alias for todo.sh

== adding tasks ==
* t add lanundry
* t add "Make peace"
* t add "Update demo +ProjectName"
   + sign : project tag
* t add "check for demo with contact @bk"
  @ sign: context tag

== listing tasks ==
* t ls
* t list
* t ls +ProjectName : list by project name
* t ls SearchKeyword
* t ls +ProjectName @bk
* t lsp : list tasks assigned with priority

== priority ==
* t pri 3 A
set priority on task #3
* t depri 2
de-prioritize task

== task completion ==
* t do 1
