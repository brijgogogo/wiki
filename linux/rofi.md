= rofi =
Rofi modes:
1. Window Switcher
2. Application Launcher
3. Desktop file launcher
4. SSH launcher
5. Script mode
6. Combi mode

== Shortcuts ==
* Shift-Delete : closes application in window switcher mode
* Shift-Return : opens application in terminal in Application launcher mode

== installation ==
sudo pacman -Ss rofi
